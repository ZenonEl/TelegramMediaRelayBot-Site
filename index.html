TelegramMediaRelayBot

Join the community of users and start using the bot today!

This is a powerful Telegram bot that allows you to download and forward media files from various platforms such as YouTube, Instagram, TikTok, and others. The bot's functionality is not limited to simply downloading media files via links. With its help, you can automate the process of downloading and forwarding media between your contacts, saving time and effort.

Get StartedDownload the latest version

Support for multiple platforms

The project uses the powerful yt-dlp tool to download media files from platforms such as YouTube, Instagram, TikTok, and many others. Thanks to integration with yt-dlp, the bot ensures stable and fast content downloading, supporting a wide range of sources and formats.

Forwarding to contacts

The bot can automatically forward downloaded media files to specified contacts or even entire groups, making it easy to share content with friends, colleagues, or communities. This is especially useful if someone cannot directly view the content by following your link. Set up a list of recipients, and the bot will independently send them media, saving your time and simplifying the file exchange process.

Proxy support

The bot supports working through proxies, including Tor, which ensures anonymity and allows bypassing restrictions imposed by platforms. This is especially useful for users who value privacy and want to protect their confidentiality.

Early development stage

The project is in the active development stage, but it already offers stable basic functionality. I am constantly working on improving and adding new features to make the bot even more convenient and powerful.

Cross-platform

The bot is designed for Linux, but thanks to the use of .NET Core, it can be adapted to work on Windows and macOS. This makes it a flexible solution for various platforms.
